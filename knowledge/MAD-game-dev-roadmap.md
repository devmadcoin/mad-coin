## PHASE 1: Programming Foundations (2-4 weeks)

### Why You Need This
Even if you use visual scripting (Blueprints, Roblox Studio), understanding code makes you 10x faster. You can debug, optimize, and build systems no drag-and-drop can touch.

### The Languages That Matter

| Language | Best For | Your Use Case |
|----------|----------|---------------|
| **Lua/Luau** | Roblox, fast scripting | Mad Phonk Awakening |
| **C#** | Unity, general purpose | Best for indie/3D games |
| **C++** | Unreal Engine, AAA | High-performance games |
| **Python** | Tools, AI, automation | Bot scripting, data analysis |
| **JavaScript/TypeScript** | Web games, tools | Browser-based $MAD games |

### Free Resources — Start Here
1. **Harvard CS50's Intro to Game Development** (FREE)
   - Uses Lua + LÖVE 2D and C# + Unity
   - 28 hours, project-based
   - https://cs50.harvard.edu/games/

2. **freeCodeCamp.org** (FREE)
   - "Learn Lua in 1 Hour" — perfect for Roblox
   - Full C# and Python tracks
   - https://www.freecodecamp.org/

3. **Codecademy** (Freemium)
   - Interactive coding in browser
   - Good for absolute beginners

## PHASE 3: Game Design Theory (Ongoing)

### Books That Change Everything
1. **"The Art of Game Design" by Jesse Schell** — The bible. Cards, lenses, psychology.
2. **"Game Programming Patterns" by Bob Nystrom** (FREE online) — Code architecture for games
3. **"A Theory of Fun" by Raph Koster** — Why games work

### Core Concepts to Master
- **Core Loops** — What players do every 30 seconds
- **Progression Systems** — Levels, unlocks, mastery curves
- **Economy Design** — Currency sinks, faucets, balancing
- **Juice** — Feedback, screenshake, particle effects that FEEL good
- **Flow State** — Challenge vs. skill balance

## PHASE 5: Monetization & Live Ops (Critical)

### Roblox-Specific (Your Current Platform)
- **Game Passes** — Permanent perks (199-999 Robux sweet spot)
- **Developer Products** — Consumables, currency (repeat purchases)
- **Premium Payouts** — Revenue share from Roblox Premium subscribers
- **Analytics** — Track DAU, session time, conversion rates
- **A/B Testing** — Test prices, icons, descriptions

### General Game Monetization
- **Cosmetic-only** — No pay-to-win (community trust)
- **Battle Pass** — Seasonal progression
- **Limited-time events** — FOMO drives engagement
- **Ad integration** — Rewarded video ads (optional)
