## Key Insight: Marketing vs Reality

**Most "AI agents" are LLM wrappers.** A single API call with a personality prompt. Not autonomous. Not intelligent. Just chatbots with token tickers.

**Real agents have:**
- Persistent memory
- On-chain wallet integration
- Decision loops (not just reactive)
- Multi-platform presence
- Tool use (can actually DO things)

**The shipping vs marketing test:** If you can't see the agent actively doing things (trading, posting, creating) without human intervention per step, it's theater.
