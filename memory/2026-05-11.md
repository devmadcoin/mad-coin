## Roblox Game Development — Mad Phonk Awakening Reincarnation Update

User shared 3 detailed spec screenshots for the MAD ROBLOX GAME reincarnation update. A hired scripter failed to deliver, so I built the entire system from scratch. Wrote 9 modular Lua scripts:

### Scripts Built
1. **ReincarnationSystem.lua** — Server-side core: reincarnation tracking, currency conversion (old → blue reincarnation currency), multipliers, item unlocks, auto-save loop
2. **ReincarnationUI.lua** — Client-side UI: tabs (Rebirth/Auras/Poses/Pets/Chests), "REINCARNATION ONLY" badges, notifications, toggle button
3. **ChestSystem.lua** — Auto-open chests at chest area, keybinds (E/T for batch open 3), chest opening animations, reward rolling, "REINCARNATION ONLY" chest tier
4. **ZoneSystem.lua** — Zone entry with restrictions (level/rebirths/reincarnations), gamepass prompt for locked zones, zone push-back, zone color coding (blue = reincarnation only)
5. **HUDSystem.lua** — Top bar currency display (coins/rebirths/reincarnations), rebirth notification, purchase buttons (coin packs + rebirth packs), MAD VIP + Ultra VIP display (Ultra VIP hidden until reincarnated)
6. **LeaderboardSystem.lua** — 3 tabs (Currency/Reincarnation/Rebirths), rank badges (gold/silver/bronze), favorite prompt, event prompt, auto-update
7. **GamepassManager.lua** — Ownership checking, purchase prompting, reincarnation-locked gamepass gating, benefit application
8. **PurchasePromptManager.lua** — Double prompt mechanic (after purchase, re-prompt appears after 2s), receipt processing, product delivery
9. **MainServer.lua** — Central init script tying all systems together

### Key Features Implemented
- **Reincarnation track** — Blue version of rebirth, unlocks at near-INF power
- **Currency conversion** — Old → blue reincarnation currency at 10% rate
- **Multipliers** — Base x2 for reincarnation, +0.5 per ultimate item
- **"REINCARNATION ONLY"** — Badges appear on poses, auras, pets, chests, zones, Ultra VIP
- **Auto-open chests** — No inventory clutter, E/T keybinds, proximity detection
- **Double purchase prompts** — "Want more? Purchase again!" appears after successful purchase
- **Ultra VIP locked** — Hidden until reincarnated
- **Mad icon turns blue** — When reincarnation track unlocked
- **Rebirth icon turns blue** — Same
- **Zone gamepass prompts** — "Unlock All Zones" prompt when entering locked zone
- **HUD overlay** — Shows coin purchase, rebirth purchase, Mad VIP purchase

### Notes
- All gamepass/product IDs are set to 0 — user needs to update with real IDs
- DataStore keys use "_v1" suffix for versioning
- Scripts are modular and can be dropped into existing project
- Auto-save runs every 60 seconds
