## The Systems

### Chaldean Numerology (Primary — Strayhorn's preference)
- **Origin:** Chaldeans of Babylonia, ~473,000 years of tradition
- **Method:** Based on sound vibration of letters, not just alphabetical position
- **Alphabet:** Cheiro's encoding (H=5, X=5, C=3)
- **Approach:** Name numbers, birth numbers, compatibility

### Pythagorean Numerology (Secondary — Strayhorn uses alongside)
- **Origin:** Pythagoras, 6th century BC
- **Method:** Alphabetical sequence (A=1, B=2...)
- **Approach:** Universal years, personal years, months, days
- **Strayhorn's view:** "Fine-tuning" addition to Chaldean, not replacement

## Core Calculations

### 1. Birth Number (Life Path)
**Method:** Reduce birth date to single digit (unless master number)

Example: February 4, 2026
- 2 + 4 + 2 + 0 + 2 + 6 = 16
- 1 + 6 = 7
- **Birth Number: 7**

### 2. Total Name Number (TNN)
**Method:** Calculate all letters in full name using Chaldean chart

**Cheiro's Chaldean Alphabet:**
```
A=1  B=2  C=3  D=4  E=5  F=8  G=3  H=5  I=1  J=1  K=2  L=3
M=4  N=5  O=7  P=8  Q=1  R=2  S=3  T=4  U=6  V=6  W=6  X=5
Y=1  Z=7
```

Example: "MAD COIN"
- M(4) + A(1) + D(4) = 9
- C(3) + O(7) + I(1) + N(5) = 16 → 1+6 = 7
- Total: 9 + 7 = 16 → 1+6 = **7**

### 3. The Goal Number
**Method:** Birth Number + TNN
- Example: 7 (birth) + 7 (TNN) = 14 → 1+4 = **5**

## Personal Year / Month / Day (Pythagorean method)

**Universal Year:** Reduce current year
- 2026: 2+0+2+6 = 10 → **1** (new beginnings year)

**Personal Year:** Birth month + birth day + universal year
- Example: Born Feb 4 → 2 + 4 + 1 = **7** (for 2026)
- **7 Personal Year:** Study, analysis, spiritual growth, inward focus

**Personal Month:** Personal year + current month
- Example: July → 7 + 7 = 14 → **5** (change, freedom month)

**Personal Day:** Personal month + current day
- Example: July 15 → 5 + 1+5 = 11 → **11** (master day, illumination)
